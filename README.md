# Basegun

Inspirations:
- https://github.com/testdrivenio/flask-vue-crud

License from https://github.com/etalab/code.etalab.gouv.fr/blob/master/LICENSE


### Install
Install make on your environment then
```bash
# install in dev mode
make build-dev

# install for prod
make build-prod
```

### Launch the website
```bash
# run in dev mode
make up-dev

# run in prod
make up-prod

# stop in dev
make down-dev
# stop in prod
make down-prod
```
